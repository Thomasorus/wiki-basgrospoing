## Introduction

- Rang : C
- Vie : 1200
  - 1400 (1 taunt)
  - 1705 (2 taunts)
  - 2050 (3 taunts)
- Stun : 72 (Longue)
- Taunt : Augmente la défense pour tout le round de 12.5% (maximum : 3
  taunts, 37.5% de bonus). Augmente aussi le recovery du Stun de 10%
  (une taunt maximum).
