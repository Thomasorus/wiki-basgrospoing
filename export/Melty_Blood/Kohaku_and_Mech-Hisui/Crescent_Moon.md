# Combos

\~ = Signifie que le combo peut commencer avec n'importe quel normal.

**BnB**

- **\~ 2C 236B-\>follow up j.AAB djc j.BC airthrow**

<!-- -->

- **\~ 2C 236B-\>follow up j.\[B\] djc j.\[C\] land 2C 5C 5\[B\] 2B 236A
  22X**

  
Possible quand l'adversaire est proche du corner ou dans le corner.
Donne un oki 22X.
